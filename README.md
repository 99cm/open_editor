SpreeEditor
===========

Rich editor for Spree. It implements different types of Rich Edotors:

- YUI Rich Editor
- WYM Editor
- TinyMCE

Example
=======

Create config/initializers/editor_config.rb with following text:

	RICH_EDITOR_IDS = %w(product_description)

And reload your server, enjoy!
